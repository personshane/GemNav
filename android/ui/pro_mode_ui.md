# Pro Mode UI (Android)

## UI Components
- Vehicle profile configuration
- Routing engine toggle (HERE/Google)
- Legal constraint indicators
- Route comparison view
- Fleet management features

## Design Principles
- Professional/commercial focus
- Clear engine status indicator
- Compliance warnings prominent
- Separate rendering for HERE vs Google

## Screens
1. Vehicle profile setup
2. Route planning with constraints
3. Engine selection toggle
4. Active navigation (HERE or Google view)
5. Route history and analytics
6. Fleet dashboard (if applicable)

## Platform Specifics
- Dual rendering pipelines
- HERE SDK MapView OR Google Maps SDK (never both)
- Commercial UI patterns
