# Legal Constraints

## Google Maps Terms
- No mixing HERE data with Google Maps tiles/UI
- Intent-based usage in Free tier
- SDK usage requires compliance with Google Maps Platform ToS

## HERE SDK Terms
- Commercial routing license required for Pro tier
- Truck routing features under commercial agreement
- Cannot display HERE routes on Google Maps UI

## Data Privacy
- On-device processing (Free tier)
- Cloud processing (Plus/Pro tiers)
- User consent requirements
