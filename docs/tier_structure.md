# GemNav Tier Structure

## Free Tier
- Gemini Nano (on-device)
- Google Maps intents only
- No API costs

## Plus Tier
- Gemini Cloud API
- Google Maps integration
- Standard routing

## Pro Tier
- HERE SDK commercial routing
- Truck-legal routes
- Google Maps toggle for car routing
- No mixing HERE data with Google Maps UI
