# AI Behaviors by Tier

## Free Mode (Gemini Nano)
- On-device processing
- Natural language to Maps intent conversion
- Limited context window
- Fast, private, offline-capable

## Plus Mode (Gemini Cloud)
- Cloud-based processing
- Enhanced context and reasoning
- Search integration
- Real-time traffic awareness

## Pro Mode (Gemini Cloud + HERE)
- Commercial truck routing
- Legal compliance checking
- Route optimization for commercial vehicles
- Toggle between truck and car routing
